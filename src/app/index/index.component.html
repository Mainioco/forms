Dynamic forms from SaaS developers to you

Ever felt like you need a dynamic creation for your forms in Angular projects but still wish the forms to be reactive? Mainio Dynamic Forms is a way to achieve this.

Features

Built for Angular 6, ready to be used instantly

It takes less code to make reactive forms now

Maintained by users who need this to SaaS development

Less than 5 minutes to achieve results

We believe that reactive forms should be easy to use but still extendable when ever needed. For this reason we created Mainio Dynamic Forms to allow us to achieve this.

As a side result we released the package as a open source to let you develop your own ideas easily.

Maintained regurally

Mainio SaaS was built using this package. As time goes on and more features are needed for the system, so does this package evolve.

Ready to start?

It's easy to get started, just go to NPM or to Github. Or if you wish, you can check out our documentation as well
